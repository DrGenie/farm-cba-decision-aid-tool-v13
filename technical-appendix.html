Quick start

This tool compares trial or demonstration treatments against a measured control using a cost benefit analysis. It turns yield and cost information into an economic comparison per hectare, shows charts, and produces exports you can use in extension notes and reporting.

  1. Download the template from the Data tab.
  2. Enter your data in Excel and save as TSV or CSV. Keep the column names unchanged.
  3. Upload the file (or paste the data).
  4. Check the upload message. If anything is missing, the tool tells you what to fix.
  5. Select the control if prompted, then review the Results tab.
  6. Export tables and charts, or use the AI ready summary prompt to draft a short write up.

Better structured data gives better analysis. If some optional fields are missing, the tool can still produce an indicative comparison using the core fields.

Minimum data required

The tool will run as long as your file contains the following:

  • At least one row marked as the control.
  • At least one row for a treatment (not control).
  • These core columns with numeric values where relevant: treatment_name, is_control, yield_t_ha, total_cost_per_ha.

You can include as many replicate rows as you like. The tool averages replicates within each treatment and compares treatment averages to the control average.

Template guide

The template contains the full set of columns supported by the tool. You do not need to delete unused columns. Leave optional columns blank if they are not available in your dataset.

Key columns

Column What to enter Example
treatment_name The label you want to appear in charts and tables. Use the same name across replicates. Control, Deep OM + PAM, Practice change A
is_control Marks the control rows. Use TRUE or FALSE (or 1 or 0). TRUE
yield_t_ha Yield in tonnes per hectare for that plot or replicate. 2.85
total_cost_per_ha Total cost in dollars per hectare for that plot or replicate. Include all relevant inputs and operations you want counted. 615.00
replicate_id Optional. A label for replicate rows, if you have it. R1, Plot 3

If you only have totals, enter them in total_cost_per_ha and leave the detailed cost columns blank.

Worked example

Below is a minimal file that will run. It includes one control and one treatment with two replicates. The tool will average the two treatment replicates and compare them to the control average.

treatment_name is_control yield_t_ha total_cost_per_ha
Control TRUE 2.60 560
Treatment A FALSE 2.90 610
Treatment A FALSE 3.10 590

In this example, the treatment average yield is 3.00 t per hectare and the treatment average cost is 600 dollars per hectare. The tool then calculates the difference from control and converts the yield difference into revenue using the grain price you enter in the Scenario tab.

How each tab works

Overview

A short introduction to what the tool does and what you need before you start.

Data

Download the template, then upload your file. After upload, the tool checks for the minimum required columns and rows. If something is missing, you will see a clear message such as a missing required column, no control identified, or no treatment rows.

Scenario

Set the grain price, time horizon, and discount rate. These settings affect the reported present value measures. If you are unsure, start with your best estimate and adjust later to see how conclusions change.

Results

Shows treatment averages, differences from control, and summary rankings. Charts and tables update immediately after upload. Replicates are aggregated automatically and results are shown per hectare.

Exports

Download results tables and a workbook version of the analysis. The exports reflect the currently loaded dataset and scenario settings.

Understanding results

The main comparison is always treatment average minus control average. A positive net benefit means the treatment performs better than the control under the price and cost settings you have entered.

  • Yield change is the treatment yield average minus the control yield average.
  • Cost change is the treatment total cost average minus the control total cost average.
  • Net benefit is the value of the yield change minus the cost change, reported over the chosen time horizon.

If your dataset is incomplete, the tool may still provide an indicative comparison using the available core fields. Treat these outputs as a starting point and improve the dataset where feasible.

Troubleshooting

My file will not load

  • Check that the first row is the header and matches the template column names.
  • Check that the file is saved as TSV or CSV, not an Excel binary format.
  • Confirm that is_control is filled for at least one row and that at least one non control row exists.
  • Confirm that yield_t_ha and total_cost_per_ha contain numbers for those rows.

I have missing values

Missing values are allowed in optional columns. If the core columns are present, the tool will run. If a core value is missing for a row, that row may be ignored in the average for that treatment.

Notes on calculations

The tool uses the uploaded yield and cost data per hectare, then applies the grain price you enter to convert yield differences into revenue differences. Present value metrics use the time horizon and discount rate you provide. Replicates are averaged within each treatment before comparisons are made.

If you need to share your results, use the Exports tab so your tables and charts match the data and scenario settings used in the tool.
