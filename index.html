Technical details (opens in new tab)

Getting started

This decision-aid supports benefit cost analysis using your own trial or demonstration data. It produces transparent comparisons between a nominated control and one or more treatments, with tables and charts suitable for reporting.

Recommended workflow: download the data template, enter one row per plot or replicate, upload the file, review any validation messages, view results and charts, then export outputs.

Minimum data to run: at least one control row and at least one treatment row, with values provided for the core outcome and cost fields used in calculations. Optional columns may be left blank.

Data quality matters. Well-structured data will produce clearer results, but the tool can still provide indicative comparisons when some optional fields are missing.

How to use this tool

  1. Download the data template, fill in your rows offline, then upload it in Data and settings.
  2. Review validation feedback and confirm the control treatment.
  3. View Results for the average treatment compared with the average control.
  4. Use Charts to see patterns visually and Exports to download outputs.

Note: if a treatment appears in multiple rows (replicates), the tool aggregates those rows and compares the average treatment with the average control.

Data tips

Common issues that stop files loading are easy to fix. Check the following before uploading.

  • Keep the first row as the header row, and do not merge header cells.
  • Use the template column names exactly, including underscores.
  • Use one row per plot or replicate. You can include as many rows as you need.
  • Mark at least one row as the control using the is_control column.
  • Avoid commas in numbers unless they are thousands separators.
  • Use consistent treatment names across rows.
